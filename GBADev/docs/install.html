Installation

System Requirements
See HAM's System requirements here: System Requirements
Installation Issues (Windows, standard edition)
The automatic installation should be a snap. Just unpack HAM, and double click the Setup.exe and follow the documentation on screen. There is currently one known issue with the installation: You cannot install HAM into a directory that has spaces in its Path, like C:\Program Files\ham ... you will need to install to a dir that has nospaces in its path. Also note that if the installer fails, you might have a very old version of the Windows Script Host, and you will need to upgrade it in order to install HAM correctly. Also see the HAM FAQ for more help.
Installation Issues (Linux and Windows Compiler Builder Ed.)
Unlike the Windows standard version, the Linux release is only available as a so-called "compiler builder" edition. This means that HAM will generate the compiler tool chain while installing, which means the installation process takes a lot more time than on Windows (on my 900MHz Athlon, it took 3 hours). The Linux edition is similar to the Win32 Cygwin compiler builder edition, and can be installed by running the "runme" shell script, which contains on-screen documentation. Also see the HAM FAQ for more help.
